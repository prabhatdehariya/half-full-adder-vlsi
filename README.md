# Design and Implementation of Half Adder and Full Adder using VLSI

## Overview
This project implements Half Adder and Full Adder circuits using Verilog HDL for digital VLSI applications.

The design was simulated and verified using VLSI design tools.

---

## Features
- Half Adder implementation
- Full Adder implementation
- Verilog HDL based design
- Simulation and waveform verification

---

## Software Used
- Xilinx Vivado
- ModelSim

---

## Verilog Modules
- half_adder.v
- full_adder.v
- top_module.v

---

## Working Principle
1. Half adder adds two binary inputs.
2. Full adder adds three binary inputs including carry input.
3. Outputs include Sum and Carry.
